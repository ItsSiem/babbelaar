# Babbelaar
De eerste Nederlandse programmeertaal!

## Kenmerken

### Functies
```bab
functie hallo(voornaam: Slinger) {
    schrijf(€"Hallo, {voornaam}!");
}
```

### Statements

#### Stel
```bab
stel mijnCijfer = 10;
```

#### Volg
```bab
volg i in reeks(1, 2) {
    schrijf(i)
}
```

### Vormen
#### Slinger
`"Hallo, wereld!"`

#### Sjabloonslinger
```bab
volg i in reeks (0, 11) {
    schrijf(€"Het getal is nu {i}");
}
```
